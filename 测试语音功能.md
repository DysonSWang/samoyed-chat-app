## 常见问题排查

### 1. 麦克风权限被拒绝
**解决**: 
- Chrome: 地址栏左侧点击 🔒 → 重置权限
- 或在浏览器设置中允许麦克风

### 2. 录音失败
**检查**:
```javascript
// 浏览器控制台测试
navigator.mediaDevices.getUserMedia({ audio: true })
  .then(() => console.log('✅ 麦克风可用'))
  .catch(err => console.error('❌ 错误:', err))
```

### 3. 上传失败
**检查**:
- OSS 配置是否正确（.env 文件）
- 网络连接是否正常
- 后端日志查看错误信息

### 4. 对方收不到消息
**检查**:
- WebSocket 是否连接成功
- 控制台查看 socket 连接状态
- 后端日志查看消息推送

## 浏览器控制台测试

```javascript
// 测试录音功能
async function testRecording() {
  try {
    const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
    console.log('✅ 麦克风访问成功', stream);
    
    const recorder = new MediaRecorder(stream);
    console.log('✅ MediaRecorder 创建成功');
    console.log('支持的 MIME 类型:', MediaRecorder.isTypeSupported('audio/webm'));
    
    stream.getTracks().forEach(track => track.stop());
  } catch (err) {
    console.error('❌ 录音测试失败:', err);
  }
}

testRecording();
```
