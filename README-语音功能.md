# 🎤 语音功能快速上手

> 像微信一样发送语音消息 - 5 分钟快速体验

---

## 🚀 快速开始（3 步）

### 1️⃣ 启动服务

```bash
cd /root/.openclaw/workspace/samoyed-chat
./start-with-voice.sh
```

或使用原启动脚本：
```bash
./start.sh
```

### 2️⃣ 访问聊天室

打开浏览器：**http://localhost:5173**

### 3️⃣ 发送第一条语音

1. 注册账号并配对
2. 点击输入框左侧的 **⋮**
3. 选择 **🎤 语音**
4. 对着麦克风说话
5. 点击 **完成** → **发送**

---

## 📱 功能演示

### 发送语音
```
点击 ⋮ → 选择 🎤 → 自动录音 → 点击完成 → 试听 → 发送
```

### 播放语音
```
点击 ▶️ 播放 → 点击 ⏸️ 暂停 → 拖动进度条跳转
```

## ❓ 常见问题

### Q: 点击录音没反应？
**A**: 浏览器请求麦克风权限时点击"允许"。如果之前拒绝了，在地址栏重置权限。

### Q: 录音上传失败？
**A**: 检查网络连接，确认 OSS 配置正确（server/.env）。

### Q: 对方收不到语音？
**A**: 检查 WebSocket 连接状态（F12 控制台），确认已配对。

### Q: 录音质量如何？
**A**: 默认 44.1kHz 采样率，清晰度高。文件大小约 10-20KB/秒。
