```
┌─────────────┐
│   产品经理   │ ← 需求分析、任务拆解、验收

       │ 任务交接

       ▼

└──────┬──────┘
       │ 完成开发
       ▼
┌─────────────┐
│  测试工程师  │ ← 功能测试、集成测试、E2E 测试

       │ 测试报告
       ▼
┌─────────────┐
│   产品经理   │ ← 验收 ✓ / 反馈 ✗
└─────────────┘
```

## 任务管理

### 任务状态

- 📋 `TODO` - 待处理
- 🔄 `IN_PROGRESS` - 进行中
- ⏸️ `BLOCKED` - 阻塞
- ✅ `DONE` - 已完成
- ❌ `FAILED` - 失败

### 任务优先级

- 🔴 `P0` - 紧急重要（立即处理）
- 🟠 `P1` - 重要（本迭代完成）
- 🟡 `P2` - 一般（下迭代完成）
- 🟢 `P3` - 优化（有空再做）

## 自动化规则

### 1. 任务自动分配
- 产品经理创建任务 → 自动分配给对应 Agent
- 任务完成 → 自动通知下一环节

### 2. 阻塞自动上报
- 任务阻塞超过 30 分钟 → 自动上报产品经理
- 连续失败 3 次 → 自动升级优先级

### 3. 质量门禁
- 代码提交 → 自动触发 Code Review
- 功能完成 → 自动触发测试
- 测试通过 → 自动触发验收

### 4. 进度监控
- 每 30 分钟自动收集各 Agent 状态
- 每日生成进度报告
- 延期自动预警
