## 🎯 功能详解

### 💬 发送文字消息

1. 在底部输入框输入文字
2. 按 Enter 键或点击📤发送
3. 支持换行（Shift+Enter）

**提示**: 
- 输入时会显示"对方正在输入..."
- 消息实时同步，无需刷新

### 🖼️ 发送图片

1. 点击📎按钮
2. 选择图片文件
3. 等待上传完成（显示进度）
4. 自动发送到聊天

**支持格式**: JPG, PNG, GIF, WebP  
**大小限制**: 最大 20MB

### 🎥 发送视频

1. 点击📎按钮
2. 选择视频文件
3. 等待上传完成
4. 自动发送到聊天

**支持格式**: MP4, WebM  
**大小限制**: 最大 20MB

**提示**: 视频支持在线播放

## 💡 使用技巧

### 快速配对

1. **生成邀请码**后，立即复制
2. 通过微信/QQ 发送给 TA
3. TA 注册后输入邀请码
4. 自动进入聊天室

### 消息管理

- **查看历史**: 向上滚动加载更早消息

| 状态 | 说明 |

| 🟢 在线 | 对方正在使用应用 |

## 🔧 常见问题

### Q1: 邀请码无效？
**A**: 检查是否输入完整 8 位，区分大小写

### Q2: 图片上传失败？
**A**: 
- 检查文件大小（<20MB）
- 检查文件格式
- 检查网络连接

### Q3: 消息发送失败？
**A**: 
- 确认已完成配对
- 检查网络状态
- 刷新页面重试

### Q4: 看不到对方消息？
**A**: 
- 确认双方都已登录
- 检查是否在同一配对关系
- 刷新页面重新连接

### Q5: 如何退出登录？
**A**: 点击右上角🚪按钮

## 🎨 自定义主题

想修改颜色？编辑 `client/src/index.css`:

```css
:root {
  --samoyed-pink: #FFB6C1;  /* 主粉色 */
  --accent-pink: #FF6B8A;   /* 强调色 */
  --bg-primary: #FFF5F7;    /* 背景色 */
}
```

修改后刷新页面即可看到效果。

## 🌟 最佳实践

1. **定期备份数据库**
   ```bash
   cp server/data/chat.db chat.db.backup
   ```

2. **清理过期数据**
   ```sql
   -- 删除 30 天前的消息（可选）
   DELETE FROM messages 
   WHERE created_at < datetime('now', '-30 days');
   ```

3. **监控 OSS 用量**
   - 登录阿里云 OSS 控制台
   - 查看存储量和流量

4. **更新应用**
   ```bash
   git pull
   cd server && npm install
   cd ../client && npm install
   ```
