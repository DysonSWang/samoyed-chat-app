# 🎤 语音功能界面说明

## 界面预览

### 1️⃣ 语音入口

**位置**: 输入框左侧附件菜单

```
┌─────────────────────────────────┐
│  ⋮  [输入框...]           ➤️   │
└─────────────────────────────────┘
     ↓ 点击 ⋮
┌─────────────────────────────────┐
│  📎 附件    🎤 语音             │
└─────────────────────────────────┘
```

---

### 2️⃣ 录音界面

**特点**: 粉色渐变主题，脉冲动画

```
┌─────────────────────────────────┐
│                                 │
│         ┌──────────┐            │
│         │  ●       │  ← 脉冲动画│
│         │  录音中  │            │
│         └──────────┘            │
│                                 │
│         0:15       ← 实时计时  │
│                                 │
│    ┌────────┐  ┌────────┐      │
│    │  取消  │  │  完成  │      │
│    └────────┘  └────────┘      │
│                                 │
└─────────────────────────────────┘
```

**视觉元素**:
- 🔴 红色圆点 = 录音中
- ⭕ 红色方块 = 已暂停
- 脉冲动画 = 正在录制
- 粉色渐变 = 萨摩耶主题色

---

### 3️⃣ 录音预览

**录音完成后可试听**

```
┌─────────────────────────────────┐
│                                 │
│      ▂ ▃ ▅ ▆ ▇   ← 波形动画   │
│                                 │
│         0:15                    │
│                                 │
│    ┌────────┐  ┌────────┐      │
│    │  ↺ 重录│  │ ➤️ 发送│      │
│    └────────┘  └────────┘      │
│                                 │
└─────────────────────────────────┘
```

**操作**:
- **重录**: 删除当前录音，重新开始
- **发送**: 上传并发送给对方

---

### 4️⃣ 语音消息（接收方）

**显示在聊天窗口**

```
┌─────────────────────────────────┐
│  ┌───────────────────────┐     │
│  │ ▶️  ▬▬▬▬▬▬○─── 0:15  │     │
│  └───────────────────────┘     │
│                          14:30 │
└─────────────────────────────────┘
```

**元素说明**:
- **▶️**: 播放按钮（点击播放）
- **▬▬▬▬▬▬○───**: 进度条（可拖动）
- **0:15**: 总时长
- **14:30**: 发送时间

---

### 5️⃣ 播放状态

**播放中**

```
┌─────────────────────────────────┐
│  ┌───────────────────────┐     │
│  │ ⏸️  ▬▬▬▬▬▬▬▬○─ 0:15  │     │
│  └───────────────────────┘     │
└─────────────────────────────────┘
```

**元素说明**:
- **⏸️**: 暂停按钮（点击暂停）
- **脉冲动画**: 播放中提示
- **进度条**: 实时前进

---

## 交互说明

### 录音流程

```
点击 🎤
  ↓
自动开始录音（脉冲动画）
  ↓
点击 "完成"
  ↓
生成预览（可试听）
  ↓
点击 "发送"
  ↓
上传 OSS + 推送给对方
```

### 播放流程

```
点击 ▶️
  ↓
开始播放（进度条前进）
  ↓
点击 ⏸️ 或播放结束
  ↓
停止/重置
```

### 进度条交互

```
点击进度条任意位置
  ↓
跳转到对应时间点
  ↓
继续播放或暂停
```

```
  ↓        ↓         ↓      ↓
```

```
   ↑        ↓

```

## 用户体验细节

### 1. 录音时
- 自动开始，减少操作步骤
- 脉冲动画提供视觉反馈
- 实时计时让用户知道时长

### 2. 预览时
- 自动试听，确认录音效果
- 重录按钮明显，方便重来
- 发送按钮突出，引导操作

### 3. 播放时
- 播放/暂停一键切换
- 进度条可拖动，支持跳转
- 播放动画提示当前状态

### 4. 错误处理
- 麦克风权限拒绝 → 友好提示
- 上传失败 → 明确错误信息
- 网络错误 → 建议检查连接

---

## 设计原则

1. **简洁**: 最少操作步骤
2. **直观**: 类似微信，零学习成本
3. **反馈**: 每个操作都有视觉反馈
4. **容错**: 支持重录、取消
5. **美观**: 萨摩耶主题，治愈系
