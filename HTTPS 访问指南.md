### Chrome/Edge 浏览器

1. 访问 `https://118.145.114.70:443/`
2. 看到"您的连接不是私密连接"警告
3. 点击 **"高级"**

### Firefox 浏览器

2. 看到"警告：潜在的安全风险"

4. 点击 **"接受风险并继续"**

### Safari 浏览器

1. 访问 `https://118.145.114.70:443/`
2. 看到"此连接不是私密连接"

## 🔄 后续升级为正式证书

如果想移除浏览器的"不安全"提示，可以申请免费证书：

### 方案 A: Let's Encrypt（需要域名）

```bash
# 1. 安装 Certbot
apt install certbot python3-certbot-nginx -y

# 2. 获取证书（替换为你的域名）
certbot --nginx -d your-domain.com

# 3. 自动续期
certbot renew --dry-run
```

### 方案 B: Cloudflare（需要域名）

1. 将域名 DNS 托管到 Cloudflare
2. 开启 SSL/TLS → Full
3. 自动 HTTPS

## 📞 常见问题

### Q: 为什么显示"不安全"？
A: 自签名证书没有经过权威 CA 认证，但加密是有效的。正式使用时建议申请 Let's Encrypt 证书。

### Q: 可以信任这个证书吗？
A: 可以。这是你自己服务器生成的证书，用于测试完全没问题。

### Q: 如何更换正式证书？
A: 参考上方"后续升级为正式证书"部分。

### Q: 手机端无法访问？
A: 确保手机和服务器网络通畅，访问 `https://118.145.114.70:443/` 并接受证书警告。
