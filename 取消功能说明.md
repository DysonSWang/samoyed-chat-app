# 🎤 语音录音取消功能

> 类似微信的上滑取消体验

---

## ✨ 新增功能

### 1️⃣ 上滑取消（类似微信）

**操作方式**:
```
录音中向上滑动 → 显示"松手取消"提示 → 继续上滑 → 松手取消录音
```

**视觉效果**:
```
┌─────────────────────┐
│                     │
│    ┌───────────┐   │
│    │  ❌       │   │  ← 大叉图标
│    │ 松手取消  │   │  ← 提示文字
│    └───────────┘   │
│                     │
│    ┌───────────┐   │
│    │  ● 录音中│   │
│    └───────────┘   │
│       0:15         │
└─────────────────────┘
```

### 2️⃣ 按钮取消

**操作方式**:
```
点击"取消"按钮 → 确认对话框 → 取消录音
```

**按钮样式**:
- 红色边框 + 渐变背景
- 悬停时放大效果
- 明确的视觉反馈

### 3️⃣ 短录音提示

**触发条件**: 录音时间 < 1 秒

**提示内容**:
```
┌─────────────────────┐
│ ⚠️ 录音时间太短     │
│ 确定要发送吗？      │
│  [取消]  [确定]     │
└─────────────────────┘
```

## 🎯 交互流程

### 上滑取消流程

```
开始录音
  ↓
手指上滑 > 100px
  ↓
显示"松手取消"提示
  ↓
继续上滑 > 150px
  ↓
松手 → 取消录音
  ↓
关闭录音界面
```

### 按钮取消流程

```
录音中
  ↓
点击"取消"按钮
  ↓
清理录音资源
  ↓
关闭录音界面
```

### 短录音确认流程

```
录音完成 (< 1 秒)
  ↓
点击"发送"按钮
  ↓
弹出确认对话框
  ↓
选择"确定" → 发送
选择"取消" → 返回预览
```

## 🔧 技术实现

### 触摸事件处理

```javascript
// 记录起始位置
const handleTouchStart = (e) => {
  touchStartY.current = e.touches[0].clientY
}

// 检测滑动距离
const handleTouchMove = (e) => {
  const currentY = e.touches[0].clientY
  const deltaY = touchStartY.current - currentY
  
  // 上滑超过 100px 显示提示
  if (deltaY > 100) {
    setShowCancelHint(true)
  } else {
    setShowCancelHint(false)
  }
}

// 判断是否取消
const handleTouchEnd = (e) => {
  const deltaY = touchStartY.current - e.changedTouches[0].clientY
  
  // 上滑超过 150px 执行取消
  if (deltaY > 150) {
    handleCancel()
  } else {
    setShowCancelHint(false)
  }
}
```

### 鼠标事件支持（桌面端）

```javascript
const handleMouseDown = (e) => {
  touchStartY.current = e.clientY
}

const handleMouseMove = (e) => {
  const deltaY = touchStartY.current - e.clientY
  if (deltaY > 100) {
    setShowCancelHint(true)
  } else {
    setShowCancelHint(false)
  }
}

const handleMouseUp = (e) => {
  const deltaY = touchStartY.current - e.clientY
  if (deltaY > 150) {
    handleCancel()
  } else {
    setShowCancelHint(false)
  }
}
```

### 资源清理

```javascript
const handleCancel = () => {
  // 停止录音
  if (mediaRecorderRef.current && isRecording) {
    mediaRecorderRef.current.stop()
  }
  
  // 关闭麦克风
  if (streamRef.current) {
    streamRef.current.getTracks().forEach(track => track.stop())
  }
  
  // 清除计时器
  if (timerRef.current) {
    clearInterval(timerRef.current)
  }
  
  // 关闭界面
  onCancel?.()
}
```

## 📊 阈值设置

| 参数 | 数值 | 说明 |
|------|------|------|
| 显示提示阈值 | 100px | 上滑超过此距离显示"松手取消" |
| 执行取消阈值 | 150px | 上滑超过此距离松手执行取消 |
| 短录音阈值 | 1 秒 | 小于此时长的录音需要确认 |

**调整建议**:
- 显示提示阈值：80-120px（根据设备调整）
- 执行取消阈值：120-180px（避免误触）
- 短录音阈值：0.5-2 秒（根据需求）

3. **触觉反馈**
   - 按钮悬停放大
   - 点击缩小
   - 明确的交互反馈
