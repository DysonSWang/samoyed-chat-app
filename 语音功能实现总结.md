## 🎯 功能特性

### 核心功能
- ✅ 浏览器原生录音（MediaRecorder API）
- ✅ 实时波形动画
- ✅ 录音时长计时
- ✅ 录音预览试听
- ✅ 重录功能
- ✅ 音频上传到 OSS
- ✅ 实时推送给配对用户
- ✅ 播放/暂停控制
- ✅ 进度条拖动
- ✅ 时长显示

### 技术实现

#### 录音流程
```
用户点击语音按钮
  ↓
请求麦克风权限
  ↓
创建 MediaRecorder
  ↓
开始录音（100ms 数据块）
  ↓
用户点击完成
  ↓
停止录音，生成 Blob
  ↓
预览试听
  ↓
用户点击发送
  ↓
上传到 OSS /api/upload/audio
  ↓
保存消息到数据库
  ↓
WebSocket 推送给对方
```

#### 播放流程
```
用户点击播放按钮
  ↓
加载音频 URL
  ↓
开始播放
  ↓
实时更新进度条
  ↓
播放结束重置
```

## 🚀 部署步骤

### 1. 确保 OSS 配置正确
```bash
cd /root/.openclaw/workspace/samoyed-chat/server
cat .env | grep OSS
```

### 2. 安装依赖（如未安装）
```bash
# 后端
cd server
npm install

# 前端
cd client
npm install
```

### 3. 启动服务
```bash
# 方式一：一键启动
./start.sh

# 方式二：分开启动
# 终端 1
cd server && npm run dev

# 终端 2
cd client && npm run dev
```

### 4. 访问测试
```
http://localhost:5173
```
