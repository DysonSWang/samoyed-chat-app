## 🐛 常见问题排查

### 问题 1: 录音没声音
**检查**:
1. 麦克风权限是否允许
2. 系统音量是否开启
3. 浏览器是否静音

**解决**:
```
浏览器设置 → 隐私和安全 → 网站设置 → 麦克风 → 允许
```

### 问题 2: 上滑取消不灵敏
**检查**:
1. 滑动距离是否足够（>150px）
2. 是否在同一垂直方向滑动
3. 触摸事件是否被拦截

**调试**:
```javascript
// 在控制台添加日志
console.log('滑动距离:', deltaY)
```

### 问题 3: 上传失败
**检查**:
1. OSS 配置是否正确
2. 网络连接是否正常
3. 文件大小是否超限

**解决**:
```bash
# 检查后端日志
tail -f backend.log
```

### 问题 4: 对方收不到消息
**检查**:
1. WebSocket 是否连接
2. 是否已配对
3. 消息是否保存成功

**调试**:
```javascript
// 检查 socket 状态
console.log('Socket 连接:', socket?.connected)
```
